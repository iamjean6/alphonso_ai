# Alphonso Performance Mentor

Alphonso is a multi-agent coaching system designed to offer elite-level sports advice and performance analysis.

## Structure
- `adk_agent`: Core agent logic and sub-agent definitions.
- `deployment`: Scripts for local testing and remote Vertex AI deployment.

## Features
- Real-time athlete identity recognition.
- Multi-agent research pipeline (Sequential Agent).
- GCS integration for performance stats analysis.
- YouTube search tool for visual demonstration finding.
